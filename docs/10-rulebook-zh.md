## 1. 遊戲簡介與目標

阿卡姆驚魂記是一款 **1–4 人合作** 的敘事型卡牌遊戲。玩家各扮演一位**調查員**,深入 1920 年代新英格蘭的詭譎事件,合力在一個個**劇本(Scenario)**中蒐集線索、推進真相、對抗由遊戲本身操控的**遭遇牌堆**與**混沌袋**。

- **合作**:沒有玩家對抗玩家;大家共同對抗遊戲。
- **敘事**:多個劇本串成一段**戰役(Campaign)**,你的抉擇、傷亡、獲得會**跨劇本延續**。
- **目標**:達成當前劇本的**勝利條件**(通常是推進到最後一「幕」);同時阻止反派的**密謀**先一步完成。輸贏都會把故事帶向不同結局 —— **失敗也是劇情的一部分**。

> 一句話:**用線索推進「幕」,搶在「密謀」累積毀滅、吞噬世界之前解開謎團。**

## 2. 組件一覽

| 組件 | 作用 |
|---|---|
| **調查員卡** | 你扮演的角色(正面能力、背面說明) |
| **玩家卡牌** | 你的牌組:支援 Asset / 事件 Event / 技能 Skill |
| **弱點卡** | 混在牌組裡的負面卡(基本弱點 + 專屬弱點) |
| **遭遇卡** | 遊戲操控的反派內容:敵人 / 詭計 / 地點 |
| **密謀卡(Agenda)** | 反派的計時器(累積毀滅 Doom) |
| **幕卡(Act)** | 玩家的進度軸(累積線索 Clue) |
| **劇本卡(Scenario)** | 該場設置、特殊規則、混沌符號效果 |
| **混沌標記 + 布袋** | 技能檢定時抽取的隨機因子 |
| **各式標記** | 線索、毀滅、傷害、恐懼、資源、充能/彈藥… |

> 電子版中,以上組件都由伺服器管理;你在畫面上看到的是**屬於你的過濾視圖**。

## 3. 核心概念

### 3.1 調查員(你)
一張大卡,定義:
- **四項技能值**:意志 **意**(Willpower)、智力 **智**(Intellect)、戰鬥 **戰**(Combat)、敏捷 **敏**(Agility)。
- **生命值 Health**(承受**傷害 Damage**)與**理智值 Sanity**(承受**恐懼 Horror**)。
- **職業 Class**:守護者 / 探求者 / 流氓 / 秘術師 / 生存者(另有中立)。
- **獨有能力**、**古老印記 ⭐ 效果**、**牌組構築需求**、必帶的**招牌卡**與**招牌弱點**。

### 3.2 玩家卡牌
| 類型 | 說明 |
|---|---|
| **支援 Asset** | 打到你檯面長期存在(武器、盟友、法術、道具);多半有**費用**、佔用**欄位** |
| **事件 Event** | 一次性效果,打出後進棄牌堆;有**費用**與觸發時機 |
| **技能 Skill** | 不「打出」;檢定時從手牌**投入(Commit)**,提供技能圖示加值 |

每張卡可能印有**技能圖示**(意/智/戰/敏 或 **百搭 wild**),投入檢定時每個相符圖示 = +1。

### 3.3 標記與資源
線索、毀滅、傷害、恐懼、資源、充能/彈藥等。**資源**是你的錢(打卡要付);**線索**推進幕;**毀滅**推進密謀。

### 3.4 混沌袋(Chaos Bag)
一袋隨機標記,是每次檢定的變數(見 §6)。難度愈高、袋內負值標記愈多。

## 4. 遊戲設置(Setup)

1. 每位玩家選一位**調查員**,備妥其**牌組**(見 §12 牌組構築)。
2. 攤開該**劇本**:依劇本說明擺放**起始地點**、放上初始**線索**(多依人數縮放,常見「每位調查員 1 個」)、組出該場的**遭遇牌堆**。
3. 擺好第 1 張**幕**與第 1 張**密謀**並排。
4. 依難度把對應的**混沌標記**放進布袋。
5. 每位調查員:洗牌 → **抽起始手牌 5 張**、取**起始資源 5**、放到起始地點。
6. 決定**首席調查員(lead)**:負責在需要「由一人決定」時拍板(不影響行動權)。

### 4.1 起始手牌調整(Mulligan)與弱點
- 抽起始手牌時**抽到弱點卡:先放到一旁,再補抽一張**(弱點不佔起手)。
- 每人可做**一次調整**:把**至多 5 張**手牌放到一旁,重抽補回到 5 張。
- 調整結束後,**放到一旁的牌(含弱點以外者)全部洗回牌庫**;先前放一旁的弱點依劇本/卡片指示處理(常見:洗回牌庫)。

> 電子版:以上由伺服器在「開打(START_SCENARIO)」時自動完成;你只需在**牌組大廳**選角、構築、按「準備完成」。

## 5. 遊戲輪與四階段

一輪固定四階段,循環到劇本結束:

```
① 神話 Mythos ─▶ ② 調查 Investigation ─▶ ③ 敵人 Enemy ─▶ ④ 整備 Upkeep ─▶(下一輪)
   (第一輪跳過①)
```

### ① 神話階段(Mythos)——第一輪跳過
1. 在當前**密謀**放上 **1 個毀滅**。
2. 檢查場上**總毀滅**是否達密謀門檻 → 是則**推進密謀**(通常是壞事;推進時**棄掉場上所有毀滅**、密謀翻面結算,揭示下一張密謀)。
3. 依座位順序,**每位調查員各抽 1 張遭遇卡並立即結算**。

**遭遇卡的結算與關鍵字:**
| 類型/關鍵字 | 結算 |
|---|---|
| **敵人 Enemy** | 依**生成(spawn)指示**放到對應地點;無指示 → 與抽到者交戰。**生成指示無法滿足 → 直接棄掉**;生成到地點視為一種「進入」 |
| **詭計 Treachery** | 立即結算卡面效果,然後棄掉 |
| **湧動 Surge** | 結算完這張後,**立刻再抽一張**遭遇卡結算 |
| **險境 Peril** | **不能與隊友討論**;只有抽到的玩家可以投入手牌做檢定,他人不得協助 |
| **冷漠 Aloof**(敵人) | 生成後**不主動交戰**(要打它得自己去交戰) |

### ② 調查階段(Investigation)——遊戲主體
- 玩家**自選順序**(合作遊戲,團隊決定誰先動;本電子版採「誰先丟就是誰的」,不強制固定輪替)。
- 輪到的調查員有 **3 個行動**,可自由組合(見 §7 行動百科)。
- 每位調查員用完行動(或選擇結束)後換下一位;全員結束 → 進入敵人階段。

### ③ 敵人階段(Enemy)
1. **獵人 Hunter** 敵人朝最近的調查員移動 1 步(若已同地點則交戰)。
2. 每個與調查員**交戰中且未耗竭**的敵人,對其發動 **1 次攻擊**(造成傷害/恐懼)。

### ④ 整備階段(Upkeep)
1. 把**調查員小卡翻回正面**(小卡翻面 = 該玩家本輪已行動完;電子版對應「✋ 我打完了」按鈕)。
2. 把耗竭(exhausted)的卡與調查員**回正(ready)**、重置行動數。
3. 每位調查員**抽 1 張牌、獲得 1 資源**。
4. **手牌上限 8 張**檢查,超過則棄置。

> **牌庫抽空**:任何時候該抽牌卻沒牌可抽 → **受 1 恐懼**,把棄牌堆洗回成新牌庫(再完成該次抽牌)。

## 6. 技能檢定(Skill Test)與混沌袋 — 全遊戲的心臟

當你「調查、戰鬥、閃避」或卡牌要求時,進行一次技能檢定。八個步驟:

1. **宣告檢定**:確定用哪項技能、難度是多少(=地點遮蔽值 / 敵人數值 / 卡牌指定)。
2. **決定修正**:套用常駐加值(如武器 +1 戰鬥)。
3. **投入技能卡(Commit)**:玩家從手牌投入**圖示相符或百搭**的卡,每個相符圖示 +1。
   - **檢定者**:可投入任意張數。
   - **同地點的隊友**:每人最多協助投入 **1 張**。
   - (本電子版「先丟先算」:收齊各方投入才抽標記。)
4. **抽混沌標記**:從袋中隨機抽 1 個,套用其修正。
5. **套用其他效果**:「抽到 X 標記則…」之類的觸發。
6. **判定成敗**:`基礎技能 + 投入 + 修正 + 標記` ≥ 難度 → **成功**,否則**失敗**。
7. **結算成敗效果**:成功/失敗各自的後果(如成功戰鬥造成傷害)。
8. **收尾**:投入的技能卡進棄牌堆;檢定結束。

### 混沌標記
- **數值標記**:`+1、0、−1、−2、−3、−4、−5、−6、−8`(依難度組合)。
- **符號標記**:
  - **⊗ 自動失敗(觸手)**:此次檢定直接失敗。
  - **⭐ 古老印記**:效果由**你的調查員**定義(通常是好事)。
  - **💀 骷髏 / 🌜 邪教徒 / 📜 石板 / 🐙 遠古種**:效果由**劇本 + 難度**定義(多為額外壞事)。
- **難度**:簡單 / 標準 / 困難 / 專家 —— 愈難,負值標記愈多。
- **進階**:封印(把標記鎖在卡上)、返還/連抽、「抽到某標記觸發能力」等。

> 電子版:混沌袋抽取在**伺服器端、可設種子**,你只知道袋內組成、無法預測抽到什麼。檢定時畫面會彈出**投入面板**,列出你可投入的卡並即時算技能值。

## 7. 行動百科(調查階段可做的事)

| 行動 | 做什麼 | 檢定 |
|---|---|---|
| **移動 Move** | 移到相連地點(首次進入會揭示並放線索) | 否 |
| **調查 Investigate** | 在所在地點找線索 | **智** vs 地點**遮蔽** → 成功取 1 線索 |
| **戰鬥 Fight** | 攻擊敵人 | **戰** vs 敵人**戰鬥值** → 成功造成傷害(=你的武器/基礎傷害) |
| **閃避 Evade** | 甩開纏住你的敵人 | **敏** vs 敵人**閃避值** → 成功使其耗竭並脫離交戰 |
| **交戰 Engage** | 主動把同地點敵人拉進你的威脅區 | 否 |
| **打出卡牌 Play** | 付費打出支援/事件 | 否(卡牌本身可能引發檢定) |
| **啟動能力 Activate** | 使用卡上「行動」能力 | 視能力 |
| **交涉 Parley** | 與特定卡/敵人互動 | 視卡 |
| **抽 1 張牌 / 獲得 1 資源** | 補牌 / 補錢 | 否 |
| **抽身 Resign** | 主動離場結束劇本(保住成果) | 否 |

> **趁隙攻擊(Attack of Opportunity)**:當你與**未耗竭**敵人交戰中,執行 **戰鬥 / 閃避 / 交涉 / 撤退 以外**的行動(移動、調查、抽牌、取資源、打非快速卡…),該敵人會**趁隙打你一下**。
> **撤退(Resign)補充**:撤退者直接離場並保住成果;**依人數縮放的判定不會因撤退而減少人數**。

## 8. 敵人與關鍵字

- 敵人有:**戰鬥值 Fight**、**生命 Health**、**閃避值 Evade**、攻擊輸出(傷害/恐懼)、**關鍵字**。
- 敵人進場多半**交戰**某位調查員(進入其**威脅區**)。交戰中的敵人在敵人階段會攻擊你。
- **打倒敵人**:把它生命扣到 0 → 被擊敗(消滅);**卡底有「勝利 X」的敵人改放到勝利點數區**(結算經驗用),而非棄牌堆。
- **甩開敵人**:閃避成功使它**耗竭並脫離交戰**;耗竭的敵人不趁隙攻擊、不主動交戰、敵人階段也不攻擊。
- **敵人階段攻擊順序**:同一敵人交戰多位調查員 → **按玩家順序**結算;多個敵人交戰同一玩家 → 由該玩家**任意順序**結算。

**常見關鍵字**
| 關鍵字 | 效果 |
|---|---|
| 獵人 Hunter | 敵人階段會朝最近調查員移動 |
| 反擊 Retaliate | 你戰鬥**失敗**時,它反傷你 |
| 警覺 Alert | 你閃避**失敗**時,它反傷你 |
| 冷漠 Aloof | 不主動交戰(要你自己去交戰) |
| 龐大 Massive | 與同地點**所有**調查員交戰 |
| 精英 Elite | 通常不受某些效果影響、無法被輕易處理 |
| 獵物 Prey / 巡邏 Patrol | 指定它優先交戰/移動的目標規則 |

## 9. 傷害、恐懼、被擊敗、創傷

- **傷害 Damage** 累積到 **≥ 生命值** 或 **恐懼 Horror** 累積到 **≥ 理智值** → 你被**擊敗**,退出本劇本。
- **傷害/恐懼分配**:你受到的傷害/恐懼可**分配給你檯面上有生命/理智的支援卡**(如盟友),但**不能溢傷**(不得分配超過該卡剩餘可承受量的多餘點數)。
- **直接傷害 / 直接恐懼(direct)**:標示「直接」者**必須由調查員本人承受**,不可分配給支援卡。
- **創傷 Trauma**:被擊敗會留下**肉體/精神創傷**,**跨劇本永久保留**(降低生命/理智上限)。創傷過重 → 調查員**陣亡**(見 §11 與電子版的死亡換角)。

## 10. 卡牌類型與檯面欄位

- **檯面欄位(Slots)**:手部 Hand ×2、秘術 Arcane ×2、盟友 Ally ×1、身體 Body、配件 Accessory、塔羅 Tarot… 同欄位有上限,超過需棄置。
- **等級 0–5**:高等卡需以**經驗值 XP** 購入。
- **獨特卡 ◆**:全場只能有一張同名獨特卡在檯面。

## 11. 戰役(Campaign)

- 多個劇本按序遊玩,串成一段戰役。
- **戰役日誌 Campaign Log**:記錄抉擇、擊殺、獲得、密語,影響後續劇本設置與分支。
- **經驗值 XP**:劇本結束依**勝利點 Victory**等結算 —— 勝利點數區的總點數**每位玩家各得等量經驗**(非均分);卡牌左上角費用下方的**白點數 = 購入所需經驗**。用來**升級牌組**(買高等卡、換卡)。
- **升級同名卡**:把牌組中的卡換成同名高等版,只需付**兩者的經驗差額**(依社群整理,細節以官方戰役規則為準)。
- **創傷**:跨劇本延續(見 §9)。
- **結局分支 Resolution**:同一劇本可能有多種結局,導向不同後續與獎懲。

> **本電子版的戰役機制**(見 [docs/09](09-lobby-save-handshake.md)):
> - 每次開打會寫一份新的**存檔版本**(不覆蓋歷史),可**跨章跳動**。
> - **中離 / 亂入**:章節邊界可有人暫時脫離、或新玩家加入;**難度隨參戰人數縮放**。
> - **接手**:戰役中只有**原玩家**能重連接手自己掉線的調查員,並重打掉線的那一動。
> - **死亡換角**:調查員陣亡 → **投票**是否讓該玩家改帶新角色;死者在此存檔**永久封鎖**。
> - **經驗**:可手動校正,但不得超過該戰役路線**至今可取得的最大經驗**。

## 12. 牌組構築(Deckbuilding)

由你的調查員定義:
- **可用職業與等級**(如探求者 0–5、中立 0–5、守護者 0–2…)。
- **牌組張數**(常見 **30 張**,不含弱點)。
- **必帶**招牌卡與招牌弱點。
- **同名卡 ≤ 2 張**;高等卡需付 XP。

> 電子版:牌組大廳的「🃏 完整選卡器」會即時驗證(合法/違規原因)、顯示每張卡的**費用與技能圖示**、並用 **XP 升級池**換入高等卡。牌組編輯也能**保存退出**(打完一章想休息)。

## 13. 快速參考:一位調查員的回合

```
你有 3 個行動,任意組合:
  🚶 移動   🔎 調查(智 vs 遮蔽)   ⚔️ 戰鬥(戰 vs 戰鬥值)   💨 閃避(敏 vs 閃避值)
  🤝 交戰   🃏 打卡(付費)         ✨ 啟動能力              🎴 抽牌 / 💰 取資源
提醒:與敵人交戰中做「會分心」的行動 → 對方趁隙攻擊你(戰鬥/閃避除外)。
線索交給「幕」推進劇情;小心「密謀」的毀滅累積。
```
